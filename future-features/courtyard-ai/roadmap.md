# Kanha Ji's Courtyard AI Roadmap

Status: 🌱 Seed Stage

The AI Companion is part of the long-term vision for Kanha Ji's Courtyard.

Development is intentionally paused because a live AI requires ongoing API costs. Rather than compromise the experience or introduce unnecessary expenses, the feature will be revisited when it becomes financially sustainable.

The philosophy, design principles, and architecture have already been completed and are preserved in this folder.

When development resumes, implementation should follow this order:

1. Review the Constitution.
2. Review the Character Bible.
3. Review the Conversation Engine.
4. Implement the Technical Architecture.
5. Build an MVP.
6. Test with real users.
7. Expand thoughtfully.

This is not an abandoned feature.

It is a planted seed waiting for the right season.
